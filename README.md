## О проекте

Репозиторий содержит материалы и код магистерской диссертации, посвящённой разработке метода оценки уровня обеспеченности населения ГОПТ


## Тема и цель
**Тема:**
Разработка метода оценки уровня обеспеченности населения городским общественным пассажирским транспортом


**Цель исследования:**  
Решение проблемы оценки уровня обеспеченности населения городским общественным пассажирским транспортом 

## Технологический стек

- `Python 3.14`
- python package: `geopandas`, `pandas`, `osmnx`, `matplotlib`, `seaborn`
- `Jupyter Notebook`
- `QGIS`

## Структура репозитория

```
pubtracc/
├── notebooks/                 # основные расчёты
│   ├── index.ipynb            # ноутбук с расчётом индекса
│   ├── kriterii.ipynb         # многокритериальная оценка маршрутов ОТ
│   └── blocksnet.ipynb        # генерация и анализ городских кварталов (необязательно)
├── data/                      # исходные и обработанные геоданные (GeoPackage)
├── results/                   # результаты анализа, таблицы, экспортированные слои
├── requirements.txt
└── README.md
```
